# Desafio de Projeto: Atualizando Relatório Financeiro com Foco na Experiência do Usuário (UX)

## Visão Geral do Projeto

Este projeto consiste na modificação e aprimoramento de um relatório financeiro existente no Power BI, com o objetivo principal de melhorar a **Experiência do Usuário (UX)**. O processo envolveu a aplicação consciente de princípios de design para tornar o relatório mais intuitivo, acessível e visualmente atraente.

O relatório final é composto por **3 páginas** de análise financeira.

---

## Princípios de Design e UX Aplicados

Para otimizar a experiência de consumo da informação, o relatório foi reestruturado levando em consideração os seguintes princípios de design:

### 1. Elementos Estruturais
* **Posicionamento:** Organização dos visuais na página seguindo o fluxo natural de leitura (topo para o rodapé, esquerda para a direita) e agrupando informações relacionadas.
* **Proporção Áurea:** Consideração da proporção ideal (aproximadamente 1:1.618) no layout para harmonização e foco nas informações mais importantes.

### 2. Elementos Visuais
* **Contraste:** Uso estratégico de cores de fundo e texto para garantir a legibilidade dos dados e direcionar a atenção para métricas-chave.
* **Estilização dos Visuais e Página:** Ajuste fino de bordas, cores de fundo e formatação para criar um ambiente limpo e profissional.

### 3. Interação e Navegação
* **Segmentação dos Dados:** Utilização de filtros e *slicers* de maneira lógica e proeminente para permitir que o usuário interaja e personalize a visualização dos dados.

> **Nota sobre as Regras:** Embora os princípios de design sejam guias, o projeto reconhece a importância de quebrar as regras de forma estratégica para adicionar criatividade e interesse ao relatório.

---

## Próximos Passos de Desenvolvimento (Navegabilidade)

Para finalizar a entrega do projeto e garantir uma UX completa, foram implementados recursos de navegação e interatividade:

1.  **Botões de Navegabilidade :** Criação de botões de navegação para transição entre as 3 páginas do relatório.
2.  **Criação dos Menus:** Implementação de menus de navegabilidade em cada página.
3.  **Estilização dos Botões:** Modificação dos botões para destacá-los e diferenciá-los visualmente, enfatizando o estado de **focalizar** (*hover*) e **selecionar**.
4.  **Consistência da Segunda Página:** Modificação da segunda página do relatório para seguir um estilo de design e UX similar ao aplicado na primeira página.
